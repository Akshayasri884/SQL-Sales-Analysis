# SQL Sales Analysis Project 🗄️

**Tools:** MySQL  
**Dataset:** Sales Data Sample (2823 rows)

## Overview
Analyzed a retail sales dataset using MySQL to extract 
business insights across products, customers, countries, and time periods.

## Key Insights
- 💰 Total Revenue: $10,032,628.85
- 🏆 Best Product Line: Classic Cars
- 👑 Top Customer: Euro Shopping Channel
- 🌍 Top Country: USA
- 📅 Best Month: November 2004
- 📦 92% orders successfully shipped

## Queries Covered
- Total Sales Revenue
- Sales by Product Category
- Top 10 Customers by Revenue
- Monthly Sales Trend
- Best Performing Country
- Average Order Value
- Sales by Deal Size
- Top 5 Products by Sales
- Year-wise Revenue Comparison
- Orders by Status
